## ğŸ”§ CUSTOMIZATION GUIDE

To adjust colors:

1. **Edit colors.css:**
   ```css
   --joy-orange: #ff6b35;  /* Change this value */
   ```

2. **Update a gradient:**
   ```css
   --gradient-sunset: linear-gradient(135deg, NEW_COLOR 0%, ...);
   ```

3. **Modify animation speed:**
   ```css
   --transition-normal: 0.3s ease-in-out;  /* Change 0.3s */
   ```

4. **Adjust shadows:**
   ```css
   --shadow-lg: 0 8px 24px rgba(0, 0, 0, 0.2);  /* Change opacity */
   ```

Changes apply globally to all components!
